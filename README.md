# College Web Development Portfolio

This repository contains all the web development projects, exercises, and assignments completed during my college coursework. It showcases a variety of HTML, CSS, and multimedia skills developed throughout the program.

## Structure

- **76776-exam.html**: Final exam or major project file.
- **websites/**: Main directory for all website projects and exercises.
	- **404.html**: Custom 404 error page.
	- **exercises.html**: Collection of practice exercises.
	- **index.html**: Main landing page for the websites folder.
	- **privacy.html**: Privacy policy example.
	- **assets/**: Shared assets (CSS, images, videos, favicons).
	- **contact/**: Contact page project.
	- **exercises/**: Themed exercises (hero image, hero text, hero video, etc.) with individual HTML and CSS files.
	- **portfolio/**: Portfolio projects, including:
		- **crypto-wallet/**
		- **fashion/**
		- **project-hero/**
		- **project-multi-column/**
		- **smoothies/**
	- **store/**: Storefront project.
	- **tw/**: Additional themed web pages.

## Technologies Used

- HTML5
- CSS3 (including custom and framework styles)
- Responsive design
- Multimedia integration (images, videos)
